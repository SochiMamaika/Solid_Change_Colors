Solid Edge Face Color Editor

Приложение для работы с Solid Edge, позволяющее просматривать грани 3D-модели и изменять их стили (цвета) с помощью удобного графического интерфейса на Windows Forms.

Описание проекта

Этот проект представляет собой Windows Forms-приложение на C#, которое подключается к открытому документу в Solid Edge и позволяет:

- Подключиться к уже открытому файлу Solid.
- Получить список всех граней модели.
- Выделять грани в интерфейсе Solid Edge при выборе их в списке.
- Назначать выбранный стиль выделенным граням.

Как использовать

1. Открыть Solid Edge

2. Запустить приложение
Программа подключается к текущему активному документу.

3. Нажать кнопку `Обновить`
- Получит доступ к телу модели и извлечёт все грани.
- Отобразит список всех граней и панель доступных FaceStyles (цветовых стилей).

4. Выделить грань в списке
- Грань будет подсвечена в Solid Edge.
- Можно выбрать несколько граней одновременно.

5. Выбрать стиль из панели справа
- Клик по кнопке стиля применяет его ко всем выделенным граням.

Структура проекта

- `Form1.cs` — основная логика: подключение к Solid Edge, обработка граней, интерфейс.
- `panelColors_Paint()` — создаёт кнопки стилей на панели.
- `add_color()` — применяет выбранный стиль к выделенным граням.
- `listViewFaces_SelectedIndexChanged()` — отвечает за выделение граней в SE.
- `btnRefresh_Click()` — обновляет список граней и стили.

Зависимости

- Solid Edge COM библиотеки (Добавляем ВСЕ с Solid Edge).
- .NET Framework (4.7.2).
